# Wemos D1 mini toolbox

## A just esp8266 with some extra modules like IR and  Radio 433.

### Features:
- Control with ~~WiFi~~ and Serial.
- Working with NECext(Epson).
- Can use Flipper-IRDB.

### Dependencies
- [IRremote]("https://github.com/Arduino-IRremote/Arduino-IRremote")
- [rc-switch]("https://github.com/sui77/rc-switch")
